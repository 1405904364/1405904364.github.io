# Distinct 去重

把查询结果去除重复记录

# 注意

  1. distinct 只能出现在所有字段的最前方。
    select name, distinct job from emp;// 错误
    select distinct name, job from emp;// 正确
    这样表示对 name 和 job 组合起来进行去重。
